## Slug Generation Behavior
- Slug is auto-generated by `KBEntryService` from the `Title` field.
- Observed pattern: kebab-case from title text (e.g., "Test Entry abc123" → "test-entry-abc123").
- Each test uses a `Guid`-suffixed unique title to avoid slug collisions.
